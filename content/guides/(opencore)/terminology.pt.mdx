| Termo        | Descrição                                                                                                                                                                                                                                                                                                                                                      |
| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| macOS        | Sistema operacional baseado em UNIX da Apple usado em máquinas Mac e "O que faz um Mac ser um Mac".                                                                                                                                                                                                                                                            |
| Windows      | Sistema operacional proprietário da Microsoft que é usado e suportado em uma ampla gama de dispositivos (fique com este sistema operacional se não quiser dores de cabeça)                                                                                                                                                                                     |
| Linux        | Família de sistemas operacionais de código aberto semelhantes ao Unix baseados no kernel Linux, um kernel de sistema operacional lançado pela primeira vez em 17 de setembro de 1991 por Linus Torvalds. O Linux é tipicamente empacotado em uma distribuição Linux. Note que embora o macOS e o Linux possam ser baseados em UNIX, eles são muito diferentes. |
| Distros      | Abreviação de Distribuições. Distros Linux são como o Linux é distribuído. No entanto, quando se trata de macOS, distros são instaladores mistos de macOS com várias ferramentas que não são da Apple. **Não use distros de macOS.**                                                                                                                           |
| Hackintosh   | O processo de instalar o macOS em um PC, note que **Hackintosh NÃO É o sistema operacional**, também pode se referir à máquina que foi "hackeada" para fazer o macOS funcionar nela. EX: _Eu instalei o macOS nesta máquina Windows, portanto tenho um Hackintosh. Mas eu NÃO instalei "Hackintosh"._                                                          |
| Bootloader   | Software que carrega um sistema operacional, geralmente feito pelos criadores do sistema operacional. O OpenCore tecnicamente não é um bootloader por si só (veja a explicação do gerenciador de boot abaixo). O Boot.efi da Apple seria o bootloader real em um Mac ou Hackintosh.                                                                            |
| Boot Manager | Software que gerencia bootloaders – temos vários destes: Clover, systemd-boot, OpenCore, rEFInd, rEFIt... Estes são geralmente vistos como preparando o sistema para o bootloader real.                                                                                                                                                                        |

---

| Termo        | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| OpenCore     | A nova sensação na cena Hackintosh, feita com segurança em mente pela [equipe Acidanthera](https://github.com/acidanthera), tem inicialização mais rápida e é mais leve que o Clover. Dá muito mais trabalho para configurar, mas também suporta muitas coisas de forma mais nativa que o Clover (como Hibernação, FileVault 2, Teclas de Atalho de Boot...).                                                                                                                                                                                                                     |
| Clover       | Um gerenciador de boot agora considerado legado com o lançamento do OpenCore. Este guia não cobrirá usos deste software.                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ACPI         | A Interface Avançada de Configuração e Energia (ACPI) fornece um padrão aberto que os sistemas operacionais podem usar para descobrir e configurar componentes de hardware do computador, mais sobre isso será discutido mais tarde no guia.                                                                                                                                                                                                                                                                                                                                      |
| DSDT/SSDT    | Tabelas em seu ACPI que descrevem os dispositivos e como o sistema operacional deve interagir com eles, por exemplo, colocando o computador em suspensão, acordando, alternando GPUs, portas USB.                                                                                                                                                                                                                                                                                                                                                                                 |
| .AML         | O formato de arquivo compilado do ACPI, e o que seu PC executará. `.DAT` é outra extensão com exatamente o mesmo uso.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| .DSL         | O código fonte para ACPI – isto é o que você edita e compila para seu computador. **NÃO** confunda este formato de arquivo com `.ASL`.                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Kexts        | Também conhecidos como **K**ernel **Ext**ensions, são os drivers do macOS. Eles são usados para realizar diferentes tarefas como drivers de dispositivos ou para um propósito diferente (no Hackintosh) como patchear o sistema operacional, injetar informações ou executar tarefas. Kexts não são a única parte de um bom Hackintosh, pois são comumente combinados com patches e correções ACPI.                                                                                                                                                                               |
| BIOS         | O Basic Input/Output System é um firmware usado para realizar a inicialização do hardware durante o processo de boot (inicialização) e fornecer serviços de tempo de execução para sistemas operacionais e programas. O firmware BIOS vem pré-instalado na placa do sistema de um computador pessoal e é o primeiro software a ser executado quando ligado (fonte: Wikipedia). É um software legado que foi feito nos anos 70 e ainda é usado até hoje devido à sua maturidade.                                                                                                   |
| UEFI         | A Unified Extensible Firmware Interface (UEFI) é uma especificação que define uma interface de software entre um sistema operacional e o firmware da plataforma. UEFI substitui a interface de firmware legada Basic Input/Output System (BIOS) originalmente presente em todos os computadores pessoais compatíveis com IBM PC, com a maioria das implementações de firmware UEFI fornecendo suporte para serviços BIOS legados. UEFI pode suportar diagnóstico e reparo remoto de computadores, mesmo sem sistema operacional instalado. (fonte: Wikipedia)                     |
| UEFI Drivers | Como qualquer outro sistema operacional, UEFI tem drivers e eles são carregados pelo Clover ou OpenCore. Eles também são destinados a carregar dispositivos ou realizar outras tarefas, como carregar drives HFS da Apple com HfsPlus.efi, patchear o `boot.efi` do macOS e assim por diante. Você pode encontrá-los como `Drivers Clover` ou `Drivers OpenCore`, todos são drivers UEFI. (Nota: use apenas drivers que são destinados a esse gerenciador de boot específico. Mais informações podem ser encontradas na [página de Conversão do Clover](){/* TODO: Add link */}). |

| Termo | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| EFI   | Pode denotar duas coisas: <br/>- Firmware do Mac, que é o mesmo que UEFI, mas bastante modificado apenas para Macs, então não tão "Universal"<br/>- A partição no seu disco rígido que armazena software lido pelo UEFI para carregar sistemas operacionais (como o bootloader do Windows) ou Aplicações UEFI (como OpenCore), é formatada em FAT32 e tem um ID tipo `EF00` (em hexadecimal). Pode ser nomeada ESP ou SYSTEM, e geralmente tem de 100MB a 400MB de tamanho, mas o tamanho não reflete em nada.                                                                                                                                                                                                                                                                                                                                                                                                                 |
| MBR   | Master Boot Record é um tipo especial de setor de boot no início de dispositivos de armazenamento em massa de computador particionados como discos fixos ou drives removíveis destinados ao uso com sistemas compatíveis com IBM PC e além. O MBR foi introduzido pela primeira vez em 1983 com o PC DOS 2.0. O MBR contém as informações sobre como as partições lógicas, contendo sistemas de arquivos, são organizadas nesse meio. O MBR também contém código executável para funcionar como um carregador para o sistema operacional instalado—geralmente passando o controle para o segundo estágio do carregador, ou em conjunto com o registro de boot do volume (VBR) de cada partição. Este código MBR é geralmente referido como um carregador de boot (fonte: Wikipedia). Este formato é usado em configurações BIOS/Legacy. O formato MBR suporta um tamanho máximo de 2 TiB e um máximo de 4 partições primárias. |
| GPT   | GUID Partition Table (GPT) é um padrão para o layout de tabelas de partição de um dispositivo de armazenamento físico de computador, como um disco rígido ou unidade de estado sólido, usando identificadores universalmente únicos, que também são conhecidos como identificadores globalmente únicos (GUIDs). Formando parte do padrão Unified Extensible Firmware Interface (UEFI) (substituição proposta pelo Unified EFI Forum para o PC BIOS), é no entanto também usado para alguns sistemas BIOS, devido às limitações das tabelas de partição master boot record (MBR), que usam 32 bits para endereçamento de bloco lógico (LBA) de setores de disco tradicionais de 512 bytes (fonte: Wikipedia). Geralmente, este é o formato de disco que você quer usar em um sistema UEFI.                                                                                                                                      |

| Termo      | Descrição                                                                                                                                                                                                                                                          |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| EC         | Controlador Embutido. Comunica-se entre a placa principal e periféricos embutidos como teclas de atalho, portas ou bateria.                                                                                                                                        |
| PLUG       | Permite que o XCPM, gerenciamento de energia Apple XNU (kernel do sistema operacional), se conecte permitindo melhor controle geral da CPU. Suportado apenas no Haswell e mais recentes.                                                                           |
| AWAC       | ACPI Wake Alarm Counter Clock, o relógio interno da placa. Contrasta com o Relógio em Tempo Real (RTC). O macOS não pode se comunicar com relógios AWAC, então eles devem ser patcheados.                                                                          |
| PMC        | Power Management Controller, em placas-mãe B360, B365, H310, H370, Z390 os OEMs esqueceram de mapear esta região e então precisam do SSDT-PMC para evitar falhas de página                                                                                         |
| PNLF       | Display de luz de fundo interno, o macOS usa este dispositivo PNLF para enviar e receber informações para controle de brilho                                                                                                                                       |
| XOSI/\_OSI | `_OSI` é usado para determinar qual sistema operacional está sendo inicializado, renomear para XOSI nos permite enganar o hardware fazendo-o pensar que estamos inicializando um sistema operacional diferente                                                     |
| HPET       | High Precision Event Timer, sistemas operacionais usam isso para determinar como se comunicar com dispositivos(IRQ). O macOS pode ser muito exigente em como os dispositivos são configurados, então às vezes precisamos patchar o HPET.                           |
| RHUB       | Root USB Hub, onde as portas USB são definidas. Se certas definições estiverem faltando aqui, as portas USB podem não funcionar no macOS                                                                                                                           |
| IMEI       | Intel Management Engine Interface, lida com tarefas diversas. No macOS, a Apple depende do IMEI para aceleração da GPU Intel. Se usar um ID desconhecido como usar um chipset série 7 com Sandy Bridge, o macOS não conseguirá encontrá-lo para aceleração da GPU. |
| UNC        | Uncore Bridge, similar a uma North Bridge, lida com muitas funções relacionadas ao cache. Muitas vezes os OEMs terão este dispositivo definido mas não funcional, o macOS é incapaz de lidar com estas situações.                                                  |
| SMBus      | System Management Bus, usado para permitir que dispositivos se comuniquem facilmente entre si.                                                                                                                                                                     |

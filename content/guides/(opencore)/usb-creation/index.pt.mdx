## Requisitos

- [OpenCorePkg](https://github.com/acidanthera/OpenCorePkg/releases), é altamente recomendado usar a versão de debug para mostrar mais informações.
- [ProperTree](https://github.com/corpnewt/ProperTree) para editar arquivos `.plist` (OpenCore Configurator é outra ferramenta, mas está muito desatualizada e a versão Mackie é conhecida por causar corrupção. **Por favor, evite essas ferramentas a todo custo!**).
- Você deve remover o Clover do seu sistema completamente se deseja usar o OpenCore como seu bootloader principal. Mantenha um backup da EFI baseada no Clover. Veja aqui o que precisa ser limpo: [Conversão do Clover](){/* TODO: Add link */}

## Instalador Online vs Offline [#online-vs-offline-installer]

O instalador offline tem uma cópia completa do macOS, enquanto os instaladores online são apenas imagens de recuperação (~500MB) que depois baixam o macOS dos servidores da Apple após inicializar.

- Offline
  - Só pode ser feito no macOS.
  - Windows/Linux não têm os drivers APFS/HFS necessários para montar um instalador completo.
- Online
  - Pode ser feito no macOS/Linux/Windows.
  - Requer uma conexão de internet funcionando via um adaptador de rede suportado pelo macOS na máquina de destino.
